# TombRoyale 纹理说明

本文件夹用于存放游戏的体素纹理贴图。

## 所需纹理列表（8 个基础纹理）

| 文件名 | 用途 | 颜色参考 |
|--------|------|----------|
| grass.png | 草地方块 | #567d46 绿色 |
| stone.png | 石头方块 | #808080 灰色 |
| wood.png | 木头方块 | #8b6914 棕色 |
| bronze.png | 青铜器/古墓砖 | #cd7f32 青铜色 |
| tomb_brick.png | 墓砖 | #4a4a4a 深灰 |
| chest.png | 宝箱 | #ffd700 金色 |
| zongzi.png | 粽子敌人皮肤 | #228b22 绿色 |
| sand.png | 沙子/海滩 | #f4e4c1 米色 |

## 纹理规格

- 尺寸：16x16 像素（像素风格）
- 格式：PNG（支持透明通道）
- 风格：Minecraft 体素风格

## 当前实现

游戏目前使用程序生成的像素纹理（Canvas 动态生成），可以直接运行。
如需替换为自定义纹理，请将 PNG 文件放入此目录，并修改 `World.ts` 中的材质加载逻辑。

## 纹理制作工具推荐

- MagicaVoxel - 体素建模和纹理
- Aseprite - 像素画编辑
- Photoshop / GIMP - 通用图像编辑
